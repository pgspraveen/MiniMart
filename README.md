# 🛒 MiniMart

MiniMart is a full-stack e-commerce web application built using the MERN stack. It provides secure user authentication, product browsing, and cart management functionalities. The application was deployed on AWS EC2 using PM2.

## 🚀 Features

* User Registration and Login
* JWT Authentication
* User Profile
* Product Listing
* Add to Cart
* Remove from Cart
* MongoDB Integration
* REST APIs
* AWS EC2 Deployment

## 🛠️ Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB Atlas
* JWT
* Axios
* AWS EC2
* PM2

## ⚙️ Installation

### Backend

```bash
npm install
node server.js
```

### Frontend

```bash
cd my-app
npm install
npm run dev
```
